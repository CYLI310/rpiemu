# RPiEmu - Raspberry Pi Emulator

A web-based emulator for Raspberry Pi that allows you to design and test circuits in a virtual environment.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/example/rpi-emu.git
   cd rpi-emu
   ```
2. **Install dependencies:**
    ```bash
    npm install
    ```
3. **Start the development server:**
    ```bash
    npm run dev
    ```
## Available Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Lints the code.
- `npm run preview`: Previews the production build.
